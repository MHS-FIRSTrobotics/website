Setup after Android Studio Installation

Open the Start Menu

Verify Git is installed

Change to the Public folder, and get the FTC SDK

Verify that you can find the ftc_app folder (a.k.a. the FTC SDK)

Copy the pristine copy to your user folder to use.

Import the FTC SDK into Android Studio

Chose the FTC SDK folder

You should see this after the above steps

Start building the DemoOpMode class

Implement the abstract methods!

You should see a big red exclamation mark when you hover over the beginning of the line, click and chose the first option, "Implement Methods". Otherwise, skip this step.
